Play Retro Game with Reinforcement Learning

Training a reinforcement learning agent to play a classic Atari game using only the game's video output. The project are based on OpenAI Gym Retro with a focus on the Implementation of Deep Q-Network (DQN).

Support for Pandemic and Epidemic Robot (SPERO)

SPERO is a robot developed by Petra Christian University to aid hospitals during the COVID-19 pandemic. It autonomously delivers essentials to patients and disinfects room using UV-C light. We designed and developed the robot from scratch. The robot runs on Raspberry Pi 4 and is equipped with IoT capabilities.

Application Monitoring System (AMS)

AMS is a parental monitoring tool to track and control children's gaming activities. It includes an Android app for parents, a desktop app for children, and an admin app for database management. Parents can set playtime limits, track activity, and receive updates, while the system automatically closes games when time expires.
